# Super-Sales-Dashboard-by-using-powerbi

This project is an interactive sales dashboard created using Microsoft Power BI to analyze Super Store sales data. The dashboard provides insights into sales, profit, customer segments, and regional performance through clear and interactive visualizations.

It helps identify sales trends, top-performing categories, and regional sales distribution, making it easier to understand business performance and support data-driven decision making.

Tools Used:
Microsoft Power BI,
Data Visualization and
Data Modeling
